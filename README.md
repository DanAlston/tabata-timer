# ⏱️ Tabata Timer

A simple Tabata-style interval timer with sound effects. Perfect for HIIT workouts, interval training, and fitness routines.

## Features

- ⏱️ **Interval Timer**: Work and rest intervals with customizable durations
- 🔄 **Round System**: Repeat intervals for a specified number of rounds
- 🔊 **Sound Effects**: Audio cues for phase changes and completion
- 📱 **Mobile Friendly**: Responsive design optimized for smartphones
- 🎵 **Customizable Settings**: Adjust work time, rest time, and rounds
- ⏸️ **Pause & Resume**: Pause the timer mid-workout
- 📊 **Visual Feedback**: Progress bar and phase indicators
- 🌐 **PWA Ready**: Install to home screen on iOS and Android

## Usage

1. **Open the app**: Visit [https://danalston.github.io/tabata-timer/](https://danalston.github.io/tabata-timer/)
2. **Configure settings**:
   - Work time (seconds)
   - Rest time (seconds)
   - Number of rounds
3. **Start workout**: Press the "開始" (Start) button
4. **Follow the cues**:
   - 🏋️ Green = Work phase
   - 😴 Orange = Rest phase
   - 🎉 Purple = Complete
5. **Control the timer**:
   - 暫停 (Pause) to pause mid-workout
   - 重置 (Reset) to start over
6. **Toggle sound**: Enable/disable sound effects as needed

## Default Settings

- Work time: 30 seconds
- Rest time: 10 seconds
- Rounds: 8

## Sound Effects

- **Start**: High beep to begin workout
- **Work Phase**: Medium beep to start work
- **Rest Phase**: Low beep to start rest
- **Complete**: Melody to signal workout finished

## Installation

### Add to Home Screen

**iOS (iPhone/iPad):**
1. Open the app in Safari
2. Tap the Share button
3. Scroll down and tap "Add to Home Screen"
4. Tap "Add"

**Android:**
1. Open the app in Chrome
2. Tap the menu button (three dots)
3. Tap "Add to Home Screen"
4. Tap "Add"

### Install as PWA

The app supports installation as a Progressive Web App for an app-like experience:
- Runs in fullscreen
- Works offline
- No browser chrome
